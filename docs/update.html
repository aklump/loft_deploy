Manual Update to 0.14

This update converts the old config style into YAML:

  1. Copy one of install/config/*.yml files to .loft_deploy/config.yml; based on the role of the local.
  2. Hand copy/paste config values from .loft_deploy/config into .loft_deploy/config.yml.
  3. Delete .loft_deploy/config.
  4. Run ldp clearcache.
  5. Run ldp configtest and adjust as necessary.
